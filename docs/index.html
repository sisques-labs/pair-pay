Getting Started

Clone the Repository

Get started by cloning the PairPay repository to your local machine.

git clone https://github.com/tu-usuario/pair-pay.git
cd pair-pay

Install Dependencies

Install all required packages using pnpm.

pnpm install

Configure Environment

Set up your environment variables with Supabase credentials.

cp .env.example .env.local
# Edit .env.local with your Supabase credentials

Setup Database

Initialize your database with Prisma migrations.

pnpm prisma migrate dev --name init

Run Development Server

Start the development server and open http://localhost:3000

pnpm dev
